**Thrylos** is a full-stack esports platform for hosting competitive tournaments, managing player registrations, handling UPI payments, and providing a seamless experience for players, organizers, and admins — built with fair play, scalability, and Indian compliance in mind.

> ⚠️ Thrylos supports **only skill-based games**. No gambling, betting, or chance-based activities.

---

## 🌐 Live Platform

- **URL:** https://admin.thrylos.in
- **Type:** Web-based esports tournament system
- **Audience:** Competitive gamers, tournament organizers, esports communities

---

## 🚀 Features

### 🎯 Tournament Management
- Create and manage skill-based tournaments
- Solo, Duo, and Squad formats
- Tournament schedules, rules, and entry fees
- Player and team data validation

### 📝 Player Registration
- Intuitive registration flow
- Dynamic form fields based on team type
- Email, phone, and Game UID validation

### 💳 Payments & Transactions
- QR-based UPI payment system
- Manual transaction verification
- Transaction ID tracking
- Payment status: Pending / Verified / Failed

### 🧾 Receipts
- Auto-generated payment receipts
- A4 print-optimized layout
- Save as PDF via browser
- Unique receipt and order IDs

### 🏆 Leaderboards & Results
- Real-time leaderboard updates
- Rank-based prize distribution
- Admin-controlled result publishing

### 🔐 Auth & Security
- Secure user authentication
- Role-based access: Admin / User
- Protected admin dashboard
- Input validation and sanitization

### 📱 UI
- Fully responsive, mobile-first
- Dark-themed modern design
- Smooth animations via Framer Motion

## 🧩 Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | React.js, TypeScript, Tailwind CSS, Framer Motion, Shadcn UI |
| Backend | Supabase (Auth, PostgreSQL, Realtime) |
| Payments | UPI QR flow with manual verification |
| Hosting | Netlify |

## 📁 Project Structure

| # | Path | Description |
|--:|------|-------------|
| 1 | `public/` | Static assets (images, icons, fonts, favicon) |
| 2 | `src/components/` | Reusable UI components (common, layout, forms, ui) |
| 3 | `src/pages/` | Route-based pages (home, auth, tournaments, dashboard, policies, 404) |
| 4 | `src/hooks/` | Custom hooks: `useAuth`, `useToast`, `useScroll` |
| 5 | `src/integrations/` | Supabase client, UPI payment logic |
| 6 | `src/utils/` | Constants, helpers, validators |
| 7 | `src/styles/` | Global CSS |
| 8 | `src/App.tsx` | Root component |
| 9 | `src/routes.tsx` | Route configuration |
| 10 | `vite.config.ts` | Vite configuration |
